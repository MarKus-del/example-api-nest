# Example api nest

## Tecnologias

+ Docker
+ NestJS
+ MongoDB

## Sobre

este repositório contém o codigo desenvolvido no video [Criando CRUD com NestJS - Code/drops](https://www.youtube.com/watch?v=cBIUOL6MFXw&t=159s) da [Rocketseat](https://www.youtube.com/channel/UCSfwM5u0Kce6Cce8_S72olg)

Este aula foi apresendado como criar um simples CRUD utilizando o framework NestJS Junto com MongoDB

Como conheci o NestJS durante a Imersão Full Cycle decidir assisti a essa video para completementar ainda mais meus conhecimentos nesse framework, e ainda por cima adicionei o docker ao projeto para que não seja necessario ter o node ou o mongodb na sua máquina

## Requesitos

+ Para rodar este reposotório e necessário ter o docker, docker compose e o git instalados na sua máquina!

## Como rodar ?

1. Para rodar essa api em sua máquina clone o repositório com o seguinte comando:

```bash
  git clone https://github.com/MarKus-del/example-api-nest
```

2. Rode o docker compose

```bash
  docker compose up
```
